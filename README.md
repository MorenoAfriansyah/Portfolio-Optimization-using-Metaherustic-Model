# Portfolio-Optimization-using-Metaherustic-Model
Portfolio Optimization with ARO Metaheuristic

## Problem Urgency
Optimasi Portofolio umumnya menggunakan metode statistika-finansial namun memiliki kekurangan dimana banyak asumsi statistika yang harus dipenuhi seperti asumsi distribusi normal, masalah skalabilitas, sensitivitas input, kendala rill non-linear untuk mendapatkan hasil yang dapat dipertanggung jawabkan keabsahannya. 

## Solution
Membuat model optimasi portofolio dengan Artificial Rabbits Optimization (ARO) 

## Data  
data harian dari periode 2022 hingga 2024 dengan emiten ADHI, ANTM, BBNI, BBRI, BJBR, BJTM, BMRI, ELSA, JSMR, ELSA, JSMR, dan TLKM.

## Research Results
Hasil dari luaran berbentuk dashboard optimisasi portofolio
